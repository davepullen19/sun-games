
# Mystery Keep Prototype

A tiny proof-of-concept for a daily deduction/riddle game inspired by classic 80s fantasy board game box art aesthetics.

## How to Run

1. Unzip the folder
2. Open `index.html`
3. Play the prototype in your browser

## Concept

Instead of traditional riddles, the player:
- reads a short scene
- detects a contradiction
- infers hidden logic
- learns the "language" of the game over time

## Why It Works

- extremely fast play sessions
- daily habit potential
- easy content generation
- social sharing ("I solved today's chamber in 1")
- scalable difficulty
